# IMDb Data Integration Project

## Overview

This project was developed for the B142 Data Integration module.

The objective was to implement a data integration pipeline using Apache Spark and IMDb datasets.

## Dataset Source

https://datasets.imdbws.com/

## Technologies

- Databricks
- Apache Spark
- Spark SQL
- Python
- Parquet

## Key Findings

- Documentary and Biography genres achieved the highest ratings.
- Drama generated the highest audience engagement.
- Movie production increased significantly after 2000.
- Most movies received ratings between 5 and 7.
